🎓 Student Management System

A simple Java-based Student Management System built using Object-Oriented Programming (OOP) concepts.
This project allows users to manage student records efficiently through a console-based application.

🚀 Features
Add new student details
View all students
Search student by ID
Update student information
Delete student record
Data handling using Java classes and collections
🛠️ Tech Stack
Language: Java
Concepts Used: OOP (Encapsulation, Inheritance, Polymorphism, Abstraction)
IDE: IntelliJ IDEA / Eclipse
Data Structure: ArrayList / Collections
